A LOGICA DA PROGRAMAÇÃO É O PRIMEIRO ASSUNTO A ESTUDAR QUANDO SE QUER APRENDER PROGRAMAÇÃO, É ENTENDER COMO O
COMPUTADOR FUNCIONA E EXECUTA INSTRUÇÕES E COM ISSO CHEGA-SE A CONCLUSÃO QUE OS COMPUTADORES ENTENDEM APENAS
LINGUAGENS DE MAQUINA E POR ISSO EXISTEM AS LINGUAGENS DE PROGRAMAÇÃO ELAS SURGEM PARA FACILITAR A CRIAÇÃO DE
PROGRAMAS E DAR INSTRUÇÕES PARA OS COMPUTADORES EXECUTAREM TAREFAS AUTOMATIZADAS E CALCULOS QUE HUMANOS NÃO CONSEGUEM
COM TANTA FACILIDADE E PRECISÃO QUANTO AS MAQUINAS.

PARA COMPRENDER LOGICA DA PROGRAMAÇÃO VOCÊ TEM QUE ENTENDER QUE OS COMPUTADORES EXECUTAM APENAS AQUILO QUE NOS
INSTRUIMOS A ELE FAZER E QUE O COMPUTADOR NÃO FAZ NADA SOZINHO O COMPUTADOR É BURRO E QUE É PRECISO ESPECIFICAR
EXATAMENTE O QUE ELE DEVE FAZER PARA EXECUTAR UMA TAREFA, ESSAS ESPECIFICAÇÕES SÃO CHAMADAS DE INSTRUÇÕES E SÃO
COM ELAS QUE OS COMPUTADORES EXECUTAM TAREFAS E AS DIFERENTES LINGUAGENS DE PROGRAMAÇÃO SÃO FERRAMENTAS PARA
RESOLVER DIFERENTES PROBLEMAS E TIPOS DE PROGRAMAS AS LINGUAGENS PODEM SER SEPARADAS POR ALTO NIVEL E DE BAIXO NIVEL

LINGUAGEMS DE ALTO NIVEL: SÃO LINGUAGEMS MAIS FACEIS DE ENTENDER POIS AS INSTRUÇÕES ESTÃO MAIS PROXIMAS A 
LINGUAGEM HUMANA, POREM AO USAR ELAS VOCÊ TEM MENOS PODER DE MANIPULAÇÃO DO PROGRAMA COMPARADA AS LINGUAGENS
DE BAIXO NIVEL. ASSIM CASO O PROGRAMA PRECISE DE OTIMIZAÇÃO PARA MELHOR DESEMPENHO VOCÊ NÃO TERA TANTO PODER
DE OTIMIZAÇÃO E DESEMPENHO QUANTO TERIA SE ESTIVESSE PROGRAMANDO EM UMA LINGUAGEM DE BAIXO NIVEL. 

LINGUAGEMS DE BAIXO NIVEL: SÃO LINGUAGEMS QUE NECESSITAM DE MAIS INSTRUÇÕES PARA O COMPUTADOR EXECUTAR AS TAREFAS ASSIM PODENDO CRIAR INSTRUÇÕES MAIS COMPLETAS E PODER FAZER CODIGOS MAIS ESECIFICOS E PRECISOS
ASSIM DANDO MAIS LIBERDADE PARA FAZER O QUE BEM ENTENDER.

NA PROGRAMAÇÃO A ORDEM DO FATORES NÃO ALTERAR O PRODUTO FINAL

COMO AS LINGUAGENS DE PROGRAMAÇÃO FUNCIONAM?

PARA EXPLICAR É PRECISO SABER QUE EXISTEM TIPOS DE LINGUAGENS DE PROGRAMAÇÃO OU SEJA UMA FORMA
DE SE COMUNICAR E INSTRUIR A MAQUINA, EXISTEM DIVERSOS TIPOS DE LINGUAGENS DE PROGRAMAÇÃO E CADA
UMA COM UMA VANTAGEM, DESVANTAGEM, E DIFERENÇAS.

COMPILED:

THE LANGUAGES COMPILED, ARE LANGUAGES WHAT NEEDED BE CONVERTED IN CODE MACHINE MANUALLY AND AFETER
GENERATE A PROGRAM IN CODE BINARY FOR BE EXECUTED IN PROCESSOR(CPU). 

INTERPRETED:

ARE LANGUAGES THAT OWN A PROGRAM THAT INTEPRET LANGUAGE AND DO THE PROCESS THAT TRANSFORMATION
FOR CODE OF MACHINE AND EXECUTE THE PROGRAM AUTOMATICALLY.
