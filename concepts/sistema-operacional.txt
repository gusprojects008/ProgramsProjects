UM SISTEMA OPERACIONAL É A JUNÇÃO DE VARIAS FUNÇÕES E COM ELAS PODEMOS REALIZAR OPERAÇÕES E OBTER RESUTADOS.
O SISTEMA OPERACIONAL É O MEIO PARA SE COMUNICAR COM HARDWARE E ASSIM CONSEGUIR REALIZAR OPERAÇÕES E EXECUTAR
INSTRUÇÕES E OBTER RESULTADOS. ISSO É O QUE O COMPUTADOR FAZ ELE COMPUTA OU SEJA CALCULA OPERAÇÕES E EXECUTA INSTRUÇÕES
COM ISSO COM O COMPUTADOR PODE CRIAR AUTOMAÇÕES E PROGRAMAS PARA RESOLVER PROBLEMAS E FACILITAR A EXECUÇÃO DE TAREFAS
MANUAIS, COM ELE TAMBEM É USADO PARA ARMAZENAR DADOS.

UM SISTEMA OPERACIONAL É CRIADO ESPECIFICAMENTE PARA INTERAGIR COM O COM O HARDWARE POR ISSO ELE É COMPOSTO POR
INSTRUÇÕES ESPECIFICAS QUE COM ELE CONSIGA REALIZAR OPERAÇÕES É POR MEIO DO SISTEMA OPERACIONAL QUE VOCÊ CONTROLA O COMPUTADOR E DITA O QUE ELE DEVE FAZER.

SABEMOS QUAL A FUNÇÃO DO SISTEMA OPERACIONAL E O QUE ELE FAZ ENTÃO O CONSTITUI ELE?

KERNEL: EM PORTUGUÊS NÚCLEO O KERNEL É A PARTE CENTRAL DO SISTEMA OPERACIONAL
É ELE QUE LIDA COM PROCESSOS GERENCIAMENTO DE MEMORIA E O FUNCIONAMENTO DO ENTRE O SISTEMA E O HARDWARE

MAS ANTES DO SISTEMA OPERACIONAL É NECESSARIO UM PROGRAMA BASICO DE GERENCIMENTO
DE HARDWARE E ASSIM PODENDO ACESSAR O LOCAL DE ARMAZENAMENTO E EXECUTAR AS INTRUÇÕES DE INICIAR O SISTEMA OPERACIONAL POR EXEMPLO

ESSE PROGRAMA BASICO QUE É USADO PARA INICIAR E CONTROLAR OS HARDWARES DO COMPUTADOR
E LIDAR COM O FUNCIONAMENTO DELES É CHAMADO DE FIRMWARE QUE FICA LOCALIZADO EM UM CHIP DE MEMORIA SEPARADO DOS OUTROS HARDWARES E É COM ELE QUE É POSSIVEL INICIAR OS HARDWARES E EXECUTAR INICIALIZAR O SISTEMA OPERACINAL.

O FIRMWARE PODE SER DE 2 TIPOS: BIOS(BASIC INPUT OUTPUT SYSTEM) E UEFI(UNIFIC EXTENSIBLE FIRMWARE INTERFACE)
CADA UM DELES TEM SUAS VANTAGENS E DESVANTAGENS ISSO DEPENDE DOS COMPONENTES DO COMPUTADOR

APOS O FIRMWARE É NESCESSARIO UM BOOTLOADER: O BOOTLOADER É O PROGRAMA NECESSARIO
PARA A POSSIBILIDADE DA INICIALIZAÇÃO DO SISTEMA OPERACIONAL ELE É RESPONSAVEL
POR DAR O PODER DE ACESSAR UM PROGRAMA COMO O SISTEMA OPERACIONAL ASSIM PASSANDO O USUARIO DO FIRMWARE PARA O SISTEMA OPERACIONAL E ASSIM PODENDO USAR O COMPUTADOR DA MELHOR FORMA.
